# ROS_Bebop2_Parrot
---

### 2019.12.23 ~ 2020.1.21 ( 프로젝트 종료일 : 2020.1.20 ) 동안 ROS(Robot Operating System를 이용한 드론 개발 교육을 받고 2주간 진행한 프로젝트 결과물입니다.

GPS와 AR마커를 이용하여 목표 위치의 물건을 가져오는 드론입니다.

(bebop2_copy1.py, GPS_turn.py 이용)


### 개발환경 : 
- Parrot사의 Bebop 2 
- Ubuntu 16.0.4
- ROS Kinetic
- python 2.7
